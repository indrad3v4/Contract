Blockchain Security

When you upload a file, it undergoes the following process:

  1. File is encrypted with AES-256 encryption
  2. Content hash is generated and stored on the blockchain
  3. Multi-signature validation is performed by network validators
  4. Upon successful validation, the file becomes part of the immutable blockchain record
